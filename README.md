# DeepSky
*DeepSky* is a Open Source JTAG Adapter based on FTDI FT2232.  
*Deepsky* is also designed JTAGKey clone.

# Supported Software
[OpenOCD][1] support ([config file](./conf/openocd/deepsky.cfg)).  


[1]:http://openocd.org/
